# Market Analysis AI Agent

An autonomous multi-agent system that performs market research, strategy generation, backtesting, and iterative optimization using an AutoResearch loop.

## Features

- Multi-agent architecture for specialized tasks
- AutoResearch loop for continuous improvement
- Market research and data collection
- Strategy generation based on research
- Backtesting of generated strategies
- Iterative optimization based on backtest results

## Getting Started

### Prerequisites

- Python 3.8+
- Required packages listed in `requirements.txt`

### Installation

1. Clone the repository
2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

### Usage

Run the main agent loop:
```
python main.py
```

## Project Structure

- `main.py`: Entry point of the application
- `agents/`: Contains specialized agent implementations
- `utils/`: Utility functions and helpers
- `config/`: Configuration files
- `data/`: Storage for market data and results
- `logs/`: Log files
